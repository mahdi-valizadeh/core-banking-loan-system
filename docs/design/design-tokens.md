# Design Tokens / توکن‌های طراحی

UI is Persian, RTL. Light grey surfaces, white cards, black text; purple
`#777BBD` reserved for primary actions and active state.

## Color Palette / پالت رنگ

```text
Primary / Brand
  --color-primary        #777BBD   (primary buttons, links, focus)
  --color-primary-hover  #6064A8   (hover - darker)
  --color-primary-soft   #E7E8F4   (soft purple background - badges)

Neutrals
  --color-black          #1A1A1A   (main text, header)
  --color-grey-900       #4A4A4A   (secondary text)
  --color-grey-500       #9E9E9E   (muted text, placeholder)
  --color-grey-200       #E0E0E0   (border, divider)
  --color-grey-50        #F5F5F7   (page background)
  --color-white          #FFFFFF   (cards, surfaces)

Semantic (status)
  --color-success        #3BA776   (Active / Pass)
  --color-warning        #C9A227   (Draft / Pending)
  --color-danger         #C8553D   (Rejected / Bug)
```
